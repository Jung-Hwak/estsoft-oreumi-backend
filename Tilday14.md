### 알고리즘

1. 효율적인 결과를 얻기 위해 어떤방법으로 해결할 것인가 공식화 해놓은것을 알고리즘이라고 한다.
2. 알고리즘의 조건
   1. 외부에서 제공되는 자료가 존재해야함
   2. 알고리즘은 최소 1개 이상의 결과를 가져야한다.
   3. 알고리즘의 각 단계는 명확해야 해서 애매함이 없어야한다.
   4. 알고리즘은 단계들을 유한한 횟수로 거친 후 문제를   
3. 알고리즘의 평가 기준
   1. 시간 복잡도 : 문제를 해결하는 데 걸리는 연산의 횟수를 세어본다.
      - 입력에 몇개 들어오느냐 에따라 얼마나 시간이 많이 걸리느냐 를 시간 복잡도 라고 한다.
      - 시간 복잡도를 표기할때 Big-O 표기법을 이용해서 나타낸다. 
   2. 공간 복잡도 : 문제를 해결하는 데 필요한 시스템 자원(메모리)의 크기를 본다.

   - 두 자연수 a를 b로 나누고 남은 나머지를 r 이라고 하면 a와 b의 최대 공약수는 b와 같고,
   - 이러한 성질을 따라 b를 r로 나누고 나머지를 R 이라고 했을때 r을 R로 나눈 나머지를,
   - 구하는 과정을 반복하여 나머지가 0이 되었을때 a와 b의 최대 공약수를 구할수있다.

### 에라토스테네스의 체 알고리즘

   - 고대 그리스 수학자 에라토스테네스가 발견한, 일정 범위에 있는 자연수들 중에서 소수를 찾아내는 방법이다.
   - 2부터 소수를 구하고자 하는 구간의 모든 자연수를 나열 한다.
   - 2는 소수 이므로 2를 제외한 2의 배수를 모두 제거한다.
   - 남아있는 수 가운데 3은 소수이므로 3을 제외한 3의 배수를 모두 제거한다.
   - 남아있는 수 가운데 5는 소수이므로 5를 제외한 5의 배수도 모두 제거한다.
   - 위의 과정을 반복하면 구하는 구간의 모든 소수가 남는다.
     
### 배열

* 배열이란 유형이 같은 여러 값들을 저ㅎ장하기 위한 저장 공간들의 묶음이며 배열은 객체의 한 종류이다.
1. 배열 참조 변수
   - 배열을 참조하기 위한 변수이다.
   - 배열 참조 변수를 선언 할때는 자료형 다음에 []대괄호를 붙여야 한다.
   - 배열 참조 변수는 같은 자료형의 배열로만 참조 할수있고, 배열은 new 로 생성하고, 초기화 할수있다.
   - 배열은 생성할때 그 크기를 지정 해줘야 한다.
   - 참조 변수를 선언하게 되면 저장할 수 있는 저장공간이 할당되고 그 저장 공간에 이름(식별자)를 지정해주고 배열에 접근할 수 있다.
   - 배열의 크기를 확인 하기 위해서 lnghth라는 필드로 배열의 크기를 확인할 수 있다.

     ``` 
### 다차원 배열

- 배열의 원소가 배열인것을 다차원 배열이라고 한다.
- 2차원 배열, 3차원 배열등이 존재 하는데 보통 2차원 배열을 많이 쓴다.
- 2차원 배열은 행과 열을 가지고 있는 2차원 평면으로 표현한다.(엑셀같은)
    ```
    int[] array1 = new int[10];             ← 크기가 10인 1차원 int 자료형 배열이다.
    int[][] array2 = new int[9][9];         ← 행과 열이 각각 9인 2차원 int 자료형 배열이다.
    int[][][] array3 = new int[7][5][3];    ← 각 방향의 크기가 7, 5, 3인 3차원 int 자료형 배열이다.
