2. 문장
   - 프로그램을 구성하는 기본단위이고 개발자가 컴퓨터에게 작업을 지시하는 단위이다
   - 문장은 반드시 세미콜론(;)으로 끝나야 한다.
   - 문장은 한 줄에 작성하지만 반드시 그런것은 아니고, 위에서 한문장씩 순차적으로 실행이 된다.
3. 키워드와 식별자
    - 프로그램에서 사용하는 단어에는 키워드와 식별자가 있다.
    1. 키워드 
       - 프로그램에서 미리 정의 되어있는 특별한 의미를 가진 단어
    2. 식별자
        - 프로그램 작성시 이름을 붙일때 사용하는 단어
        - 변수명(variable names)과 함수명(function names)이 식별자에 해당
        - 영문자와 숫자, 밑줄 문자(_)와 달러 기호($)를 조합해서 작성해야한다.
        - 숫자로 시작할수 없고, 공백문자는 포함할 수 없어서 공백 문자는 구분자로 쓰인다.
        - 대문자와 소문자를 구분하고 키워드는 식별자로 사용할 수 없다.
4. 괄호 작성 규칙
   - 괄호(()), 중괄호({}), 대괄호([]) 
   - 모든 괄호는 쌍으로 이루어져 있고 괄호의 짝이 맞지 않으면 오류가 발생한다. (열었으면 닫아야 한다)
5. 공백 문자

9. 자료형
    - 프로그램에서 다루는 값(data)의 종류
    - 자료형에 따라 프로그램에서 값을 다루는 방법이 다르다.
    1. `string` : 문자열 `"Hello World"`
       - 문자들의 나열, 문자들의 집합을 나타내는 자료형,JavaScript 프로그램에서 가장 많이 다루는 자료형
       - 문자열은 연결 연산자(+)로 문자열을 연결 할수있따
       -  작은 따옴표(') 혹은 큰 따옴표(")로 묶어서 표현할 수 있다 작은 따옴표로 시작했으면 끝도 작은 따옴표 여야한다. 혼용x
        ``` 
        1. "This is a "string"."        // 오류
        2. 'This is a 'string'.'        // 오류
        3. 'This is a "string".'        // 문자열
        4. "This is a 'string'."        // 문자열
    2. `number` : 숫자(정수, 실수) `78, 3.14`
       1. 특별한 값들이 존재한다 Infinity 무한대를 의미하는 키워드이고, 숫자를 0으로 나누면 연산결과가 Infinity가 된다.
       2. NaN 자료형은 number이지만, 숫자로 나타낼 수 없는 값을 의미하는 키워드(Not-a-Number)
    3. `boolean` : 불린(참, 거짓) `true, false`
    4. `function` : 함수 `function () {}`
    5. `object` : 객체 `{}`
    6. `undefined` : 선언되지 않았거나 할당되지 않은 값
       - "존재 하지 않는 것" 을 표현 하는 자료형
         1. 변수로 선언 하지 않은 식별자
         2. 변수로 선언은 했지만 초기화 되지않은 변수
    7. 자료형 검사
            ```

            ```
10.  연산자

        - 나머지 연산 자란 나누고 나서 남은 값을 구하는 연산자 이다.
            ```

     - 프로그램에서 문장 들은 차례대로 실행이 된다.

        ```
11.  피연산자의 개수에 따른 연산자의 분류
    1. 단항 연산자 : 피연산자가 하나인 연산자
        1. +num : 부호 연산자 / 변수의 값을 양수 변환
        2. -num : 부호 연산자 / 변수의 값을 음수로 변환
        3. ++num : 증가 연산자(전위) / 변수의 값에 1을 더함
        4. num++ : 증가 연산자(후위) / 변수의 값에 1을 더함
           - 전위 연산 
              - 증가,감소 연산자가 피연산자 앞에 쓰이는 경우
              - 전위 연산을 먼저 수행하고, 그 다음에 전위 연산을 제외한 다른 연산을 수행 
           1. --num : 감소 연산자(전위) / 변수의 값에 1을 뺌
           2. num-- : 감소 연산자(후위) / 변수의 값에 1을 뺌
              - 후위 연산
                - 증가,감소 연산자가 피연산자 뒤에 쓰이는 경우
                - 후위 연산을 제외한 다른 연산을 먼저 수행하고, 그 다음에 후위 연산을 수행
            ```
                var num = 10;  
                console.log(num);   // 10
                console.log(++num); // 11
                console.log(num++); // 11
                console.log(num);   // 12
     2. 이항 연산자 : 피연산자가 두 개인 연산자
     3. 삼항 연산자 : 피연산가자 세 개인 연산자
12.  비교 연산자와 논리 연산자
     1. 비교 연산자
          - 연산자 양변에 오는 값을 비교해서 참이면 true가 연산 결과가 되고, 거짓이면 false가 연산 결과가 된다.
          1. `==` : x와 y의 값이 같은지 비교(동등한지)
          2. `!=` : x와 y의 값이 다른지 비교
          3. `===` : x와 y의 자료형과 값이 같은지 비교(일치한지)
          4. `!==` : x와 y의 자료형과 값이 다른지 비교
          5. `>` : x가 y보다 큰지 비교
          6. `<` : x가 y보다 작은지 비교
          7. `>=` : x가 y보다 크거나 같은지 비교
          8. `<=` : x가 y보다 작거나 같은지 비교
     2. 논리 연산자
          - 피연산자를 참 또는 거짓으로 다루고, 연산 결과도 참 또는 거짓이 되는 연산자
          1. `!` : 논리 부정 연산자 / 피연산자가 참이면 거짓, 거짓이면 참
          2. `&&` : 논리곱(AND) 연산자 / 피연산자가 모두 참이면 참 아니면 거짓
          3. `||` : 논리합(or) 연산자 / 피연산자 중에서 하나다로 참이면 참, 모두 거짓이면 거짓
     3. 조건 연산자
        - 조건에 따라 다른 표현식을 연산 결과로 취하는 연산자
        - 연산자 중에 유일하게 피연산자가 세 개인 연산자(삼항 연산자)
        - 첫 번째 피연산자의 값에 따라, 참이면 두 번째 피연산자가 연산 결과가 되고, 거짓이면 세 번째 피연산자가 연산 결과가 된다.
        - 즉, 조건 연산의 결과는 두 번째 피연산자 또는 세 번째 피연산자이다.
        ```
        // var x = 3; 
        // var y = 7;     max = x > y ? x : y;
        // var max;            // x > y 표현식이 참이면 x가, 거짓이면 y가 표현식의 결과다 된다.
    ``` 

14. 조건문과 반복문

        ```

        ↑--- 들여 쓰기를 해서 이 문징이 if 구문에 묶인 문장임을 나타낸다.       Condition -----→ false
                                                                            ↓             |
                                                                        Conditional_code  |
                                                                            ↓             |
                                                                            End    ←---------
    ```
        -조건문의 예시
        var score = 86;
        // var score = 58;

            console.log('합격입니다!');
