## HTML에 대하여(VSCode 이용)
기본적으로 workspace 파일을 만들고 그곳에 HTML문서를 작업할 공간을 만들어 두는게 좋다.  
VSCode로 HTML 문서를 만들때  
파일이름뒤에 확장자는 .html 로 해야한다.  
ex) `day01.html`


HTML(Hyper-Text Markup Language) 문서의 기본 구성 과 요소
- `<!DOCTYPE html>` HTML의 문서의 형식을 정의 하는 태그로서 HTML 문서의 최상단에 와야함.  
- `<html lang="ko"> </html>` 작성할 HTML 문서의 주된 언어(주로 쓰는 그 나라의 언어) 
- `<meta charset="UTF-8">` HTML 문서의 문자 인코딩 방식을 유니코드의 UTF-8로 지정하는 태그
- `<head></head>` HTML 문서의 정보 등을 표현하는 태그
- `<body></body>` HTML 문서의 구조(콘텐츠)를 표현하는 태그
- HTML 문서를 작성시 각 태그들은 꺽인 괄호로`<태그>` 시작하여 `</끝 태그>`로 끝난다.
- HTML 5 표기법 부터 내부에 콘텐츠를 가지지 않으며 태그에 따라 특정한 역할을 하는 태그들이 있는데. 그것들은 시작태그 만 가지고있다 예시`<br>`,`<hr>`이 있다.

HTML에서의 중요한 태그
1. `<!DOCTYPE html>`,
2. `<html lang=”그 문서의 주된 언어”>`,
3. `<head>`(html 태그의 요소로 들어 가야함),
4. `<meta charset="UTF-8">`(head 태그의 요소로 들어 가야함),
5. `<body>`(html 태그의 요소로 들어 가야함)

HTML 요소 
- HTML 요소가 동작하는데 필요한 정보를 제공하거나 추가정보를 부여하고,
- 내부 텍스트를 가지지 않는 HTML 요소도 속성을 사용한다.
- HTML 속성의 이름은 소문자로 작성하는것이 원칙이다. 예시 `<h1></h1>`
- HTML 요소는 중첩 될수있고 중첩은 가능하지만 교차는 될수없다.  
중첩 예시`<ul> <li> </li> </ul>` 교차 예시 `<ul> <li> </ul> </li>`
- 프로그래밍 언어로 작성한 코드들의 설명을 적기위해 주석(Comments) 를 사용하는데  
주석은 프로그램 실행에 있어 어떠한 영향을 미치지 않고 작성하는 프로그래밍 언어의 설명을 적기 위해 사용가능하다 `<!-- -->`(HTML문서의 주석)

- `<head></head>` 요소에 포함되는 `<meta>, <title>, <link>, <style>, <script>`요소
- `<body></body>` 요소에 들어가는 콘텐츠를 표현하는 요소를 담는 컨테이너(containers)를 수행하는 요소는
- `<ul>, <ol>, <table>, <tbody>, <tr>, <form>, <pre>, <div>`등이 있고,  
화면에 표시되는 콘텐츠를 표현 하는 요소는 `<h1>, <p>, <a>, <strong>, <code>, <img>, <input>`요소 등이 있다.

HTML 문서의 들여쓰기  
- 웹 브라우저는 HTML문서, 스타일 시트, JavaScript 프로그램에 있는 공백문자(White Spaces; 빈칸, 개행 문자, 탭 등)를 해석하지 않고 무시한다.  
HTML문서의 들여쓰기  
- HTML 문서에서 중첩(형제) 관계나 종속(후손) 관계 등을 이해하기 편하도록 들여쓰기를 한다.
- 들여쓰기는 HTML 문서의 중첩 관계를 쉽게 이해할수 있게 끔 도와준다. 무조건 해야하는것은 아니지만 프로그램을 작성해서 나중에 디버깅등 다른 프로그래머와 작업을 할때 프로그램의 가독성을 올려준다

### HTML의 글자 태그
1. 제목 태그
- `<h1>, <h2>, <h3>, <h4>, <h5>, <h6>` 
- 1부터 6까지의 숫자로 지정된 6단계의 제목 수준을 가진다. `<h1>` 태그는 문서 전체에 사용되는 제목이고 `<h2>` 는 그 아래 수준이며 이러한 식으로 `<h6>`태그까지 계속된다.
2. 본문 태그
- `<p>, <br>, <hr>`
3. 앵커 태그
- `<a>`
- HTML(Hyper-Text Markup Language)에 나오는 Htper-Text는 사용자의 선택에 따라 특정한 정보와 관련된 부분으로 이동할 수 있도록 조직화된 문서를 의미한다.
- HTML 문서가 이렇게 조직화된 문서 형태를 가질 수 있는 이유가 앵커 태그이다.
- 앵커 태그란 서로가 다른 HTML 문서로 이동하거나 문서 내부의 특정한 위치로 이동할때 쓰인다.
- 앵커 태그는 href 속성에 url(`#` 혹은 HTML 요소의 `id`)을 속성 값으로 사용(지정)한다. 
- 문서 내부이동을 원할때는 HTML 요소에 `id` 속성을 설정하여 이동한다
4. 글자 형태
- `<b>, <i>, <strong>, <em>, <small>, <sub> <sup>` 등
- HTML 문서에서 가장 큰 비중을 차지하는 태그들 이다.
- HTML 문서에는 다양한 콘텐츠가 들어가지만, 그 중에서 텍스트의 비중이 가장 크다.
