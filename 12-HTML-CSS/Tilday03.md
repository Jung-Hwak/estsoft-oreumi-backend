# 오늘 배운 내용

## HTML/CSS

* 선택자
    1. 속성 선택자
        -[attribute] 지정한 HTML 속성이 있는 HTML 요소를 선택하는 선택자 
    2. 가상 클래스 선택자
        - 특정 상태에 있는 요소를 선택할 수 있는 선택자
    3. 상태 선택자
        - 입력 요소에 상태에 따라서 선택할 수 있는 선택자
    4. 구조 선택자
        - 형제 관계에 있는 요소들 중에서 특정 위치에 있는 요소를 선택할 수 있는 선택자
        - 구조선택자에서 :nth-child 에 뒤에는 수열이 올수있다.
            - 수열중에서 :nth-child(3n) 에서 n 은 0부터 1씩 증가 하는 정수 이며, 따라서 0, 3, 6, ... 등을 선택한다

    5. 형태 구조 선택자
        - 태그 형태를 구분해서 형제 요소들 중에서 특정위치에 있는 요소를 선택
        - `:first-of-type`

    6. 링크 선택자
        - href 속성을 적용 해놓은 앵커 요소에 적용되는 선택자

    7. 부정 선택자
        - 주어진 선택자로 선택되지 않은 요소를 선택함
        - `:not(li)`

    8. 가상 요소 선택자
        - 요소가 아닌 특정 영역을 HTML 요소인 것처럼 선택해준다.
        - 가상 요소 선택자 는 :: 콜론을 두개를 붙인다.

    9. 전후 문자 선택자
        - 지정한 요소의 첫번째 자식 요소 이전이나 마지막 자식 요소 이후에 가상 요소를 자식 요소로 추가해준다.
        - 가상 요소는 content 속성을 지정 해줘야한다. 지정되어 있지 않으면 추가 되지 않는다.
        - `::before`, `::after`
        - 전후 문자 선택자인 CSS로 컨텐츠를 추가하는 것은 적절하지 않다, HTML 요소를 꾸며주기 위하여 써야한다.
    10. 반응 문자 선택자
        - 사용자가 드래그 하였을때 그 드래그 한 영역을 선택할 수 있다.
        - `::selection`

    * 효율적인 선택자
        - 선택자를 선택할때 범위가 좁은 선택자가 더 좋은 선택자 이며, 전체 선택자는 사용하지 않아야한다.
        - 클래스 선택자보다는 아이디 선택자가 더 효율적이고, 태그 선택자보다는 아이디 선택자가 더 효율적이다.
        - 후손 선택자 보다는 자식 선택자가 더 효육적이며 상속을 적극적으로 사용해야한다.

    * CSS 스타일 속성들의 단위

        - CSS 크기 단위 `px (픽셀)`, `pt (포인트)`, `% (백분율 단위)`, `em (배수 단위)`,`rem (배수 단위)` `vh, vw(브라우저 화면 높이나 너비의 백분율)`
        - `%`는 스타일이 적용된 속성의 크기를 기준으로 하고, `em` 과 `rem` 은 [글자 크기]를 기준으로 한다. `rem`은 브라우저의 글자 크기를 기준으로 한다.
        - `vh, vw`는 브라우저의 화면을 기준으로 한다

1. CSS 박스 모델
    - 모든 HTML 요소는 박스 모양으로 표시된다.
    - 모든 박스 모델은 테두리(border)가 존재하고 테두리와 () 사이에 패딩(padding)이 존재하고 제일 안쪽은 컨텐트 영역(content box) 이며, 제일 바깥쪽인 테두리 바깥은 마진(margin)이있다.
    1. widht, heigth 
        - widht 와 heigth 는 요소의 크기를 지정 하는 속성이 아닌 요소의 콘텐트 영역의 크기를 지정 하는 스타일 속성이다.
    2. margin, padding
        - margin 
        - padding
        - margin 속성과 paddin 속성은 상, 우, 하, 좌, 방향의 값을 따로 지정 할수있고, 값이 4개가 주어지면 위쪽(top)을 기준으로 시계방향으로 적용이 된다.
            - ex) margin(padding): (1.)1px 2px | (2.)1px 2px 3px 1 (3)1px 2px ← [1(위쪽, 아래쪽 = 1px), (우, 좌 = 2px)], [2(위쪽 = 1px), (우, 좌 = 2px)],(아래쪽 = 3px ), [3(위쪽 = 1px) (아래쪽 = 2px)]
            - ex) margin(padding): 1px 2px 3px 4px 
        - 상, 우, 하, 좌, 방향의 값을 따로 따로 지정이 가능하다
            - ex) margin(padding): 1px 2px

2. HTML 요소의 공간 차지 방식
    - 블록(Block)형식, 인라인(Inline), 인라인 블록(Inline-Block) 형식들이 있다.
        - 블록 형식은 HTML 요소가 컨테이닝 블록 너비 전체를 상자 형태이며 안쪽 여백(padding)과 바깥쪽 여백(maring)을 가졌다.
        - 블록형식은 
        - 인라인 블록 형식은 인라인 형식의 특징을 기본으로 하며, 블록 형식의 특징을 같이 가지고있고
        - 문장 내에 
    - 인라인 형식 안에 블록 형식의 컨텐츠는 들어갈수 없지만 블록 형식안에 인라인 형식은 들어갈 수 있다.
    * 컨테이닝 블록(Containing Blocks)
        - HTML 요소가 화면에 표시될때 컨테이닝 블록을 기준으로 배치가 되며, 컨테이닝 블럭은 position 스타일 속성에 따라서 결정이 된다.(position 스타일의 기본값은 `static`이다.)
        - position 이 주어지면 그 요소와 가장 가까운 블록형식의 조상 요소의 콘텐트 영역이 된다.
    * box-sizing 스타일 속성
        - width height 속성이 어떤 영영의 크기를 설정할지 지정할 수 있게끔 하는 속성
        - `content-box` : 박스 모델에서 width, height 속성이 콘텐트 영역의 크기를 지정 (기본 값으로 지정 되어있음)
        - `border-box`  : 박스 모델에서 width, height 속성이 테두리를 포함한 영역의 크기를 지정함.
3. 문서의 일반적인 흐름
    1. 블록 형식(`<div>, <h1>, <p>`, ...)
        - HTML 요소가 컨테이닝 블록 너비 전체를 상자 모양으로 차지함
        - 안쪽 여백 padding, 바깥쪽 여백 margin을 가진고, 스타일 속성으로 위치 여백 너비 높이등을 설정할 수 있다.
    2. 인라인 형식(`<span>, <a>`, ...)
        - HTML 요소가 문장 내에서 텍스트 처럼 영역을 차지함
        - 줄을 바꾸지 않고 다른 인라인 요소나 내부 텍스트와 함깨 한 행에 위치하고, 상하 외부 여백marin을 가지지 않고 너비 높이를 설정할 수 없음.
    3. 인라인 블록 형식(`<img>, <input>`, ...)

    * 컨테이닝 블록이 화면에 표시될때
        - HTML 요소들은 화면에 표시 될때 공간을 차지하는 방식이 존제 하는데 표시 형식에따라 달라지며, 좌상단을 기준으로 배치가 된다.
        - 블록 형식은 좌상단을 기준으로 왼쪽 아래로 차근차근 배치되고, 인라인 형식은 좌상단을 기준으로 오른쪽으로 배치된다.
        - 블록 형식의 요소는 크기를 지정 하지 않으면, 컨테이닝 블록의 너비를 채운다.
