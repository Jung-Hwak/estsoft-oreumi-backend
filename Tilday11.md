7. 키워드 식별자
   - 키워드는 반드시 정의된 의미를 가지는 단어를 써야한다
     - `this, class, in ,throw, with, int` 등이 있다.
   - 식별자 
     - 프로그램에 이름을 붙일때 사용 하는 단어이며 영어로 써야한다.
     - 대 소문자를 구분하고 숫자로 시작할수 없고, 공백 문자 또한 포함할 수 없다.
   - `클래스` = 각 단어의 첫 글자는 대문자로 해야하고 파스칼 표기법으로 작성하는 것이 관례이다.
   - `변수,메서드` = 소문자로 시작하고 두번째 단어부터는 첫 글자를 대문자로 한다
   - `기호 상수` = 상수는 모든 글자를 대문자로 해야한다.

    - Java 프로그램은 객체들로 구성이 되어있다.

9.  메인 메서드
    - Java를 실행했을때 가장 먼저 실행되는 특별한 메서드이다.
    - 일반적으론 main 메서드의 실행이 끝나면 프로그램의 실행또한 끝난다.
    - Java 애플리케이션에는 반드시 main 메서드가 있어야한다.
    - main 메서드를 포함하는 클래스는 public 클래스 이어야한다.
10. 메서드 
    - 입력을 받아서 작업을 수행하는 프로그램들의 집합 이다.
    - 내부 문장들을 차례대로 실행한 후에 작업 결과를 외부로 반환해준다.
    - 메서드를 실행하는 것을 메서드를 호출한다고 표현하고 메서드를 호출하면서 필요한 값을 인자 라고한다.
11. 자료형과 변수
    - 프로그램에서 다루는 값의 종류이며 java에서는 변수에도 자료형이 있다.

    1. 원시 자료형

      - 문자 = char = 'A'
        - Java 프로그램에서 문자열은 String 클래스의 인스턴스 이다

      - 정수형 숫자 = byte, short, (기본형)`int`, long = 381
      - 실수형 숫자 = float, (기본형)`double` = 3.141592
      - 논리 = `boolean` = true

12. 연산자
        ```
        x       =       2;
        ↑       ↑       ↑
        피연산자 연산자   피연산자
        y = 3;
        sum = x + y;
                ↑ 연산자
    - 대입 연산자 : `=` 변수와 같은 저장공간에 값을 저장하는 연산자, 우변에 있는 값을 좌변에 있는 변수와 같은 저장 공간에 저장한다.
    - 산술 연산자 : `+, -, *, /, %` 어떤 수(변수)등의 더하기, 빼기, 곱하기, 나누기, 나머지 등의 기능을 하는 연산자
      - 산술 연산자의 자료형은 피연산자들의 자료형은 같아야 한다(double 자료형과 int 자료형은 계산하지 못한다.)
      - 만약 피연산자들의 자료형이 다르다면 컴파일러는 두 피연산자들의 자료형을 맞추려고 시도한다.(`묵시적인 형 변환` 이라고 한다.)
    - 나머지 연산자 : `%` 어떤 수의 배수를 구하 거나 홀수 짝수 등을 구분할 수 있는 연산자
