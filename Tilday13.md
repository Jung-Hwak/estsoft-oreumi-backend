## Java

1. 중첩 반복문
   - 반복분 안에 또다른 반복문이 들어가는 것을 중첩 반복문이라고 한다.
   - for 구문에 또다른 for 구문을 사용할때 바깥쪽 인덱스 변수와 안쪽 인덱스 변수는 다르게 사용해야한다.
   - 두 반복문에서 같은 인덱스 변수를 가지면 무한 반복에 빠질수있으니 조심하자. 
   - 중첩 반복문은 2차원 구조를 표현 할수있다.
2. 반복문에서 실행 흐름을 제어 
   - 반복문의 실행 흐름을 건드리(제어)는 구문이다.
   - break와 continue 문을 너무 무분별하게 쓰면 프로그램의 실행의 흐름을 따라가기 어려워진다. 
   2. break문
     - switch 구문이나 반복문에서 break 문을 쓸수있다
     - 반복문 or switch 들이 실행이 쭉 되다가 break 문을 만나면 제어문을 빠져나가게 된다.
   3. continue문
     - 반복문에서 continue 문을 사용할수있다.
     - 반복문에서 continue 문을 만나면 현재 반복을 멈추고 다음 반복으로 실행된다.

## java.lang.String 클래스의 repeat 메서드
  
  - 인자로 주어진 횟수만큼 문자열을 반복해서 문자열을 생성해주는 메서드이다.
  - 인자가 0 이면 빈 문자열("")을 생성하고,
  - 인자가 음수이면 IlleagalArgumentException 예외가 발생하며,
  - 인자가 1이면 문자열을 그대로 반환 해준다.
  * Java 11에서 추가된 `java.lang.String`클래스의 메서드이다.
