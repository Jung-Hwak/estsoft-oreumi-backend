3. 제어문
    1. if 구문
       - 조건이 참인지 거짓인지에 따라 실행되는게 달라지는 구문이다.
       - 조건식에 따라서 실행하는 흐름이 달라지는 제어문이다.
       - 조건문에서 실행될 문장이 여러개면 중괄호로 묶어놓는다 그것을 우린 블록 이라고 한다. 
    2. if-else 구문
        - 처음 조건에 거짓이 나왔을때 나오는 문장에 쓰이는 구문이다.
        - if 다음에는 참 또는 거짓을 확인하기 위한 조건식이 나오고
        - if 절이나 else 절에는 문장 하나가 나온다.
        - else 절은 이미 조건을 가지고 있다 if키워드 다음에 나오는 표현식이 거짓인 경우 else절 의 문장이 실행된다.
        - else 절은 단독으로 나올 수 없고, 반드시 if 절이 쓰이고 else 절이 필요하면 쓰인다.
    3. if-else-if 구문
        - 조건문 안에 조건문을 중첩 하는것을 중첩 조건문 이라고 한다.
        - 중첩 조건문은 경우의 수가 많을때 쓰기 적합하다.
    4. switch 구문 
        - switch 다음으로는 변수가 나와야 하는데 정수 자료형, 문자 자료형, 문자열 자료형의 변수만 와야한다.
        - case 라는 키워드를 여러개 써서 변수에 대입되어있는 상수 값을 쓰고 `:(콜론)`을 찍는다
        - 콜론을 찍으면 이름표(라벨, 레이블)라고 한다.
        - case의 변수들이 다 참이 아니라면 default:로 이동하게 된다.
        - 실행이 되다가 break; 구문을 만나면 switch 구문을 빠져 나가게 된다.
4. 반복문
    * 반복문을 쓸때 조심해야될 것은 조건식을 잘못 지정하면 무한 반복문이 되어버린다, 그러므로 조건식을 잘 정해야한다.    
    1. while 구문
        - 가장 기본적인 반복문이다
        - while 키워드 다음으로는 조건식이 나온다.
        - 조건식의 값이 참(true)인동안 반복문은 계속해서 실행된다.
    2. do-while 구문
        - 먼저 문장을 실행하고 조건식을 검사하는 반복문이다.
        - while 구문은 조건식을 확인하고 조건식이 참이면 실행되지만 do-while 구문은 
        - 문장을 먼저 실행하고 조건식을 확인하기에 무조건적으로 문장이 한번은 실행이된다.
    3. for 구문
       - 어떠한 문장을 몇번 반복할지 정할때 쓰이는 구문이다.
       - 표현식은 생략이 가능하지만 세미콜론(;)은 생략할 수 없다.
       - while 구문은 조건에 비중을 두고 있지만, for 구문 같은 경우에는 반복횟수에 비중을 두고 있다.
       - for 구문에서는 인덱스 변수 라는 일정하게 변하는 값을 표현하는 변수를 사용한다
       - 인덱스 변수는 i > j > k >... 영어 알파벳 순서로 한다.
