## 오늘 배운 내용

### JavaScript

1. 조건문
    1. if-else (중첩 조건문)
       - 조건문에 조건문을 중첩한 경우   
    2. if-else-if 구문 
       - 중첩 조건문의 else 절에 조건문을 넣고 개행과  조건문을 사용할수있다.
    3. whild 구문
       - 가장 기본적인 반복문(loop)
       - 주어진 조건이 참인동안 문장을 반복해서 실행
       - 조건이 참인 동안 계속해서 문장을 실행
    4. for 구문
        - while 구문은 조건에 비중을 두지만, for 구문은 반복 횟수에 비중을 둔다.
2. 배열
    - 유형이 같은 여러 값들을 저장하기 위한 저장 공간들의 집합(묶음)이다.
    - 배열 리터럴([])로 생성하고, 변수에 대입할 수 있다.
    - 배열에 저장되어있는 원소들의 갯수를 확인 하기위해선 length로 가능하다.
    ```
    var numbers = [1, 2, 3, 4, 5, 6, 7, ...];  // 배열을 생성하여 변수에 저장
    console.log(numbers[1]);    // 변수 numbers에 저장한 배열중 인덱스가 1인 변수에 접근하여 출력
- 배열에 포함되어있는 원소에 접근하기 위해서 반복문(for)으로 원소에 하나씩 차례대로 접근하는 방법이 있다.
